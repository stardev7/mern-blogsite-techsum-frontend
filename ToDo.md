# Mern-Blogsite-Techsum

## Purpose of Project

This site was developed by integrating various web technologies. This site implemented user management and posts management sent by users.

Technologies Used:
> MERNStack
> Firebase
> PostgreSQL
> TailwindCSS
> Redux
